This is a Basic Video Conferencing Web Application.
<br>
Tools which i have used :
<ul>
  <li>NodeJs</li>
  <li>ReactJs</li>
  <li>AWS Chime</li>
</ul>

Basic Features :
<ul>
  <li>Create Meeting</li>
  <li>Join Meeting</li>
  <li>Screen Sharing</li>
  <li>Different Meeting Controlls</li>
  <li>Video Recording </li>
  <li>Collaborative Whiteboard</li>
</ul>
